Dark Mode

Tailwind CSS v4 Dark Mode Updates

Dark Mode Configuration in v4





// In tailwind.config.js:
module.exports = {
  // In Tailwind v4, darkMode can be:
  // 1. 'media' (default) - Uses prefers-color-scheme media query
  // 2. 'class' - Manually toggle dark mode with a class
  // 3. ['class', '[data-mode="dark"]'] - Use multiple selectors
  darkMode: 'class',
}

Tailwind CSS v4 has more flexible dark mode configuration options. The new selector format allows targeting custom data attributes or other selectors for dark mode.

New Dark Mode Strategy with Multiple Selectors

In v4, you can specify multiple dark mode selectors: 



// Example configuration:
darkMode: ['class', '[data-theme="dark"]']

// This allows either of these to trigger dark mode:
// 1. .dark class on html element
// 2. [data-theme="dark"] attribute on any parent
