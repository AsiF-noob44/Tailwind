Tailwind CSS v4 Columns Updates

Responsive Columns with Container Queries

Tailwind CSS v4 supports container queries with columns:

First Content Block

This content will flow into multiple columns based on the container's width, not just the viewport width. Container queries provide more flexibility than traditional media queries.

Second Content Block

Resize the browser to see how the columns adjust based on the container's width. This allows for more complex and responsive layouts.

Third Content Block

With container queries, components can adapt to their available space regardless of the viewport size.

Fourth Content Block

The columns will change from 1 to 2 to 3 as the container grows wider.

The @container directive creates a container query context, while @[size]: prefixes apply styles conditionally based on container width.
