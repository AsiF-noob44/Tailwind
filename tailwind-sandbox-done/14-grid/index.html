Tailwind CSS v4 Grid Updates

Subgrid (New in v4)

Tailwind CSS v4 adds support for CSS subgrid:

Parent Grid Header
This aligns with parent
This spans 2 columns and aligns with parent grid
Parent column

grid-cols-subgrid and grid-rows-subgrid allow child grid layouts to inherit track sizes from parent grid.

Named Grid Lines with Arbitrary Values

Tailwind CSS v4 makes it easier to work with named grid lines:

Main content area (spans from main-start to sidebar-start)
Sidebar area (spans from sidebar-start to main-end)

Arbitrary values in v4 make it easy to define and use named grid lines and areas.
