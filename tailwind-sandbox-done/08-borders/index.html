Tailwind CSS v4 Border Updates

Logical Border Properties (New in v4)

Tailwind CSS v4 uses logical properties for borders that respect reading direction:

v3 Properties:

border-l-4 (left border)
border-r-4 (right border)

v4 Logical Properties:

border-s-4 (start border)
border-e-4 (end border)

In v4, border-s and border-e replace border-l and border-r for better RTL (right-to-left) language support. The old properties still work but logical properties are preferred.

Enhanced Outline Controls

Tailwind CSS v4 has improved outline controls:

Border Radius Improvements

rounded-s-lg

v4 supports logical border radius (rounded-s, rounded-e, rounded-ss, rounded-se)
