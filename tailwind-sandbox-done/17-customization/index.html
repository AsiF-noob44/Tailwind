Tailwind CSS v4 Customization Updates

Key Configuration Changes in v4

// tailwind.config.js
module.exports = {
  // Content paths (new in v3+, still used in v4)
  content: ['./src/**/*.{html,js}'],
  
  // Dark mode strategy (v4 adds support for multiple selectors)
  darkMode: ['class', '[data-mode="dark"]'],
  
  // Theme customization
  theme: {
    // Container queries (new in v4)
    containers: {
      xs: '20rem',   // 320px
      sm: '24rem',   // 384px
      md: '28rem',   // 448px
      lg: '32rem',   // 512px
      xl: '36rem',   // 576px
      '2xl': '42rem' // 672px
    },
    
    // Simplified color palette (v4)
    colors: {
      // Core colors with different shades
      blue: { /* shades */ },
      // Can use extend to add to default colors
    },
    
    // Extend existing theme
    extend: {
      // Add custom animations
      animation: {
        'spin-slow': 'spin 3s linear infinite',
      }
    }
  }
}

New Customization Features in v4

Container Query Configuration:

Tailwind v4 allows customizing container query breakpoints directly in the config. 

containers: {
  xs: '20rem',
  sm: '24rem',
  // ...other breakpoints
}

Multiple Dark Mode Selectors:

Tailwind v4 supports multiple dark mode activation strategies. 

darkMode: [
  'class', 
  '[data-mode="dark"]',
  '@media (prefers-color-scheme: dark)'
]
