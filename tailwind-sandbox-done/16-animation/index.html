Tailwind CSS v4 Animation Updates

Animation Direction & Fill Mode

Tailwind CSS v4 adds control over animation direction and fill mode:

Normal direction:

animation-direction-normal

Reverse direction:

animation-direction-reverse

Alternate direction:

animation-direction-alternate

Fill none:

animation-fill-none

Fill forwards:

animation-fill-forwards

Fill both:

animation-fill-both

Animation Timeline (New in v4)

Scroll down to see animation

The animation is tied to the scroll position of this container.

The animation-timeline-scroll utility ties animations to scroll position.

The animation-timeline-scroll and animation-range-entry utilities allow animations to be driven by scroll position.

Motion-safe animation:
