CSS-First Configuration

In Tailwind CSS v4, configuration moves from JavaScript to CSS:

Theme CSS Variables

This uses var(--color-blue-500)
This uses var(--color-emerald-400)

All theme values are available as CSS variables

CSS Configuration Example

@import "tailwindcss";

@theme {
  --font-display: "Satoshi", sans-serif;
  --breakpoint-3xl: 1920px;
  --color-brand-100: oklch(0.99 0 0);
  --color-brand-500: oklch(0.84 0.18 117.33);
  --ease-fluid: cubic-bezier(0.3, 0, 0, 1);
}

@utility btn {
  padding: 0.5rem 1rem;
  border-radius: 0.25rem;
  font-weight: 500;
}

@custom-variant hocus (&:hover, &:focus) {
  /* Creates a custom variant for hover+focus */
}

Configure theme values, create utilities, and define variants directly in CSS
